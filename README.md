# Role Composer

# Example Playbook

- hosts: all
  vars:
    composer_version: 1.6.5
    composer_checksum: sha384:544e09ee996cdf60ece3804abc52599c22b1f40f4323403c44d44fdfdd586475ca9813a858088ffbc1f233e9b180f061
    
  roles:
    - role: blunix.role-composer

```
